🚀 TaskPro

TaskPro is a full-stack task management application built using the MERN Stack (MongoDB, Express, React, Node.js) with TypeScript and Tailwind CSS.

It helps users efficiently manage daily tasks, track progress, organize projects, and boost productivity through a modern and responsive interface.

Context API / Redux (if used)

🔹 Backend

MongoDB (Mongoose ODM)

JWT Authentication

bcrypt (Password Hashing)
